HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The section element allows us to organize content on our page based on the topics we have.
  • HTML provides content and a basic framework for all pages on the web

CSS

  • A margin indicates how much space we want around the outside of an element
  • A margin also assigns properties to the top and bottom of a section
  • A padding indicates how much space we want around the content inside an element
  • A class selector is a way to style a section to differentiate a topic using the term card.
  • A box-shadow is defined on an X and Y axis
  • Using auto in code assigns properties to the left and right in a section

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it
  • git pull origin main pulls the latest main branch
  • Remember to sync your work with GitHub by using the add, commit, pull, and push commands in git

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A "falsy" statement consists of: 0, -0, On, " ", null, NaN, or a Boolean data type of false.
  • All other statements are considered "truthy" statements.
  • Strings need to be in quotes. Numbers do not need to be in quotes.
  • The first item in any array has an index of 0, not 1.
  • Index, otherwise known as location identifiers, will always increase by 1 increment.
  • Points about "for loops": 1. Keyword is "for", 2. three-part conditional statement, 3. code block.
  • After declaring a function, you must call it at the spot you want it to execute.
  • You can test the functionality of a funtion in the Chrome Dev Tools.
